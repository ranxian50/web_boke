# 用户操作指南

## 访客操作

### 浏览文章
1. 打开博客首页（http://localhost:5173）
2. 在文章列表中点击感兴趣的卡片查看文章详情
3. 文章详情页会显示完整内容、标签、阅读量和评论区

### 按标签筛选
1. 首页顶部有标签筛选栏，显示所有可用的标签
2. 点击某个标签（如"Java"），页面会仅显示带有该标签的文章
3. 点击"全部"恢复显示所有文章

### 发表评论
1. 在文章详情页底部的评论区，填写昵称（必填）、邮箱（选填）和评论内容
2. 点击"发表评论"按钮提交
3. 评论会立即显示在文章下方
4. 点击其他评论的"回复"按钮可以回复该评论（嵌套回复）

### 查看随机推荐
- 首页底部有"猜你喜欢"板块，会随机展示3篇文章推荐给访客

## 管理员操作

### 登录
1. 访问 http://localhost:5173/login
2. 输入用户名和密码（默认：admin / admin123）
3. 登录成功后自动跳转管理后台

### 仪表盘
- 管理后台首页，展示文章总数、标签数等统计信息

### 发布文章
1. 在侧边栏点击"发布文章"
2. 填写文章标题
3. 编写文章内容（支持两种方式）：
   - **直接编写**：在文本框中直接写 Markdown
   - **上传文件**：点击"上传 .md / .ipynb 文件"按钮上传本地文档
4. 选择内容类型（Markdown / Jupyter Notebook）
5. 选择发布状态（直接发布 / 存为草稿）
6. 点击"发布文章"按钮
7. 系统会自动：
   - 提取前80字作为摘要
   - 调用 DeepSeek API 分析内容并生成技术标签

### 管理文章
1. 在侧边栏点击"文章管理"
2. 查看文章列表（标题、类型、阅读量、发布时间）
3. 点击"编辑"按钮修改文章
4. 点击"删除"按钮移除文章

### 管理标签
1. 在侧边栏点击"标签管理"
2. 查看所有标签及其关联的文章数量
3. 标签由 AI 自动生成，也支持手动创建

### 管理评论
1. 在侧边栏点击"评论管理"
2. 查看和删除访客评论

### 退出登录
- 点击管理后台右上角的"退出登录"按钮

## 常见问题

### Q: 如何修改管理员密码？
当前版本暂未提供密码修改功能。如需修改，可直接在数据库中更新 `tb_user` 表的 password 字段（使用 BCrypt 加密）。

### Q: AI 标签功能需要什么？
需要在 `application.yml` 中配置有效的 DeepSeek API Key。
即使不配置，文章发布功能也不受影响，只是不会自动生成标签。

### Q: 上传文件有大小限制吗？
默认限制为 50MB，可在 `application.yml` 中调整。

### Q: 部署到生产环境需要注意什么？
1. 修改 JWT Secret（`jwt.secret`），使用高强度的随机字符串
2. 修改管理员默认密码
3. 禁用 JPA 的 `ddl-auto: update` 改为 `validate`
4. 配置 HTTPS
5. 设置合适的 CORS 域名白名单
